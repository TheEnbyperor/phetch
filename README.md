# phetch

dirt simple terminal gopher client.

## features

- small (<1MB), single file executable for linux and macos
- technicolor design
- no nonsense keyboard navigation

## development

    cargo run
    
## resources

- [rfc 1346](https://tools.ietf.org/html/rfc1436)
